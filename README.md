# create-react-tailwind-site

Landing page for [@oudom/create-react-tailwind](https://www.npmjs.com/package/@oudom/create-react-tailwind) — a CLI tool to create a React + TypeScript + Tailwind CSS project in one command.

## Live

[crt.oudom.dev](https://crt.oudom.dev)

## Built With

- React
- TypeScript
- Tailwind CSS v4
- Vite

## Run Locally

```bash
git clone https://github.com/oudomm/create-react-tailwind-site
cd create-react-tailwind-site
npm install
npm run dev
```
